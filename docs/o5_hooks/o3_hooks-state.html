whats-a-hook

  • What is a Hook? A Hook is a special function that lets you “hook into” React features. i.e. useState
  • When would I use a Hook?use state. function component && hook || class

declaring-a-state-variable

  • What does calling useState do?It declares a “state variable”.

    Normally, variables “disappear” when the function exits but state variables are preserved by React.

  • What do we pass to useState as an argument?The only argument to the useState() Hook is the initial state. number, stirng, object vs object in class
  • What does useState return?It returns a pair of values: the current state and a function that updates it. const [count, setCount] vs this.state.count && this.setState
